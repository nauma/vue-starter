# vue-starter

> A Vue.js webpack project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# list dependencies
- vuex
- vue-router
- vuex-router-sync
- vue-meta
- vue-cookie
```
